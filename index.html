Interview Bot

A modern, fully responsive AI Interview Practice platform where users can practice real interview questions and get instant AI-powered feedback. Features a clean hero page, one-question-at-a-time interview flow, answer submission with live feedback, and smooth animations throughout. The backend serves questions via a REST API (GET /start, POST /answer) and returns intelligent feedback on every answer — all running beautifully on a premium, mobile-friendly UI.

AI News Agent

An autonomous AI agent that fetches live news every 5 minutes, sends it to OpenAI, and automatically transforms it into polished blog content — then posts it directly to your blog without any manual work.
